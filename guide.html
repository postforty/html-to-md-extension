📖 HTML to MD Saver 사용 설명서

웹 페이지의 원하는 영역을 클릭 한 번으로 깔끔한 마크다운(Markdown) 문서로 변환하고 번역할 수 있는 강력한 도구입니다.

기본 사용법

  • 브라우저 상단의 퍼즐 아이콘이나 고정된 HTML to MD Saver 아이콘을 클릭하여 팝업을 엽니다.
  • [요소 선택 시작] 버튼을 누릅니다.
  • 웹 페이지에서 마우스를 움직여 추출하고 싶은 영역(문단, 이미지, 표 등)에 다크 오렌지색 점선 테두리가 생기면 클릭합니다.
  • 자동으로 마크다운 변환(및 번역)이 진행되며 미리보기 창이 뜹니다. [다운로드]를 누르면 .md 파일로 저장됩니다.

⌨️ 핵심 단축키 (고급 기능)

Shift + 마우스 클릭 (다중 선택)
여러 개의 흩어진 문단이나 요소들을 하나의 문서로 스크랩하고 싶을 때 사용합니다.
Shift 키를 누른 상태로 요소들을 클릭하면 굵은 오렌지색 실선으로 선택이 고정됩니다. 원하는 요소들을 모두 선택한 후 화면 하단에 나타나는 [선택된 N개 요소 변환하기] 버튼을 누르면 한 번에 병합되어 추출됩니다.
ESC (선택 취소 및 종료)
요소 선택 모드를 켰다가 취소하고 싶거나, 다중 선택을 초기화하고 원래 웹 페이지 상태로 돌아가고 싶을 때 언제든지 ESC 키를 누르면 안전하게 종료됩니다.

🌐 AI 번역 기능 (Gemini)

확장 프로그램 설정(설정 버튼 클릭) 메뉴에 Google Gemini API 키를 등록하면 강력한 AI 번역 기능을 사용할 수 있습니다.

  • 번역본만 저장: 원본 텍스트 대신 자연스러운 한국어 번역본으로 깔끔하게 대체됩니다.
  • 원본과 번역본 함께 저장: 각 문단별로 한국어 번역본이 먼저 나오고, 바로 아래에 원본 영어 문장이 그대로 유지되어 어학 공부나 원문 대조에 매우 유용합니다.
