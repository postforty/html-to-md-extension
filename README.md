# HTML to MD Saver (크롬 확장 프로그램)

웹 페이지에서 원하는 요소를 마우스로 클릭하면 해당 영역을 추출하여 **마크다운(Markdown, `.md`)** 파일로 즉시 다운로드해 주는 크롬 확장 프로그램입니다.

## ✨ 주요 기능
- **직관적인 요소 선택:** 확장 프로그램을 실행하고 웹 페이지 위를 마우스로 이동하면 선택될 영역이 시각적으로 하이라이트됩니다.
- **다중 선택 지원 (Shift + Click):** `Shift` 키를 누른 상태로 요소를 클릭하면 여러 개의 요소를 동시에 스크랩하여 하나의 파일로 병합할 수 있습니다.
- **안전한 취소 모드 (ESC):** 영역 선택 중 언제든 `ESC` 키를 누르면 요소 선택 모드를 즉시 안전하게 종료할 수 있습니다.
- **AI 기반 자동 번역:** Gemini API와 연동하여 추출된 마크다운을 한국어로 자동 번역할 수 있으며, 원본과 번역본을 교차하여 동시에 저장하는 강력한 모드를 지원합니다.
- **GFM(GitHub Flavored Markdown) 지원:** 기본 마크다운 문법뿐만 아니라 **표(Table)**, 취소선 등의 GFM 문법을 완벽하게 지원합니다.
- **코드 블럭 보존:** 웹 페이지 내부의 소스 코드(`<pre>`, `<code>` 등)의 줄바꿈과 포맷팅을 훼손 없이 ` ``` ` 형태의 펜스 코드 블럭으로 깔끔하게 유지합니다.
- **절대 경로 자동 변환:** 원본 웹 페이지에서 상대 경로(`.`, `/` 등으로 시작)로 작성된 링크(`<a>`)와 이미지(`<img>`) 주소를 클릭 가능한 절대 경로(Absolute URL)로 자동 변환하여 링크 깨짐을 방지합니다.
- **한글 깨짐 방지:** 마크다운 변환 후 파일을 다운로드할 때 Base64 인코딩을 적용하여 한글 및 특수문자 깨짐을 원천 차단합니다.
- **안전한 스크립트 주입:** 캡처링 이벤트를 활용해 기존 페이지의 클릭 동작이나 링크 이동을 방지하며, `mousedown` 기본 동작 차단으로 텍스트 드래그(블록 지정)를 방어합니다.
- **최신 규격 준수:** 구글 크롬 확장 프로그램의 최신 표준인 **Manifest V3** 규격을 기반으로 개발되었습니다.

## 🚀 설치 및 사용 방법 (개발자 모드)

1. 이 저장소를 로컬 컴퓨터에 다운로드하거나 `git clone` 합니다.
2. 크롬 브라우저를 열고 주소창에 `chrome://extensions/`를 입력하여 **확장 프로그램 관리 페이지**로 이동합니다.
3. 우측 상단의 **'개발자 모드'** 토글 버튼을 켭니다.
4. 좌측 상단에 나타난 **'압축해제된 확장 프로그램을 로드합니다' (Load unpacked)** 버튼을 클릭합니다.
5. 다운로드한 `html-to-md-extension` 폴더를 선택하여 확장 프로그램을 등록합니다.
6. 마크다운으로 스크랩하고 싶은 웹 페이지로 이동합니다. (단, `chrome://` 내부 페이지나 크롬 웹스토어에서는 보안상 작동하지 않습니다.)
7. 브라우저 우측 상단의 퍼즐 모양 아이콘을 눌러 **HTML to MD Saver**를 실행하고 **[요소 선택 시작]** 버튼을 클릭합니다.
8. 화면에서 원하는 영역을 마우스로 클릭하면 `extracted-element.md` 파일이 다운로드됩니다!

## 📂 프로젝트 구조
- `manifest.json`: 크롬 확장 프로그램의 설정 파일 (Manifest V3)
- `popup.html` / `popup.js`: 확장 프로그램 팝업 UI 및 스크립트 주입 로직
- `content.js`: 화면 요소 선택, 하이라이트, HTML 추출 및 마크다운 변환 커스텀 룰 로직
- `background.js`: 다운로드(Data URI, Base64) 처리 워커
- `turndown.js`: HTML을 Markdown으로 변환해 주는 핵심 라이브러리
- `turndown-plugin-gfm.js`: 표(Table) 등의 부가 문법을 지원하는 GFM 확장 라이브러리

## ⚠️ 한계점 및 주의사항
- 크롬 확장 프로그램의 보안 특성상 페이지 내부에 삽입된 외부 `<iframe>` (예: 유튜브 영상, 외부 결제창, 소셜 위젯 등) 내부의 요소는 선택하거나 변환할 수 없습니다.
- 코드를 수정하신 경우 반드시 확장 프로그램 관리 페이지(`chrome://extensions/`)에서 새로고침(↻) 아이콘을 누르고, **테스트 중인 웹 페이지(F5)도 함께 새로고침해야** 수정된 스크립트가 정상적으로 반영됩니다.
