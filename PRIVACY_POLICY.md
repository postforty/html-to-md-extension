# HTML to MD Saver 개인정보처리방침 (Privacy Policy)

**시행일:** 2026년 7월 4일

`HTML to MD Saver`(이하 "확장 프로그램")는 사용자의 개인정보를 매우 중요하게 생각하며, 크롬 웹 스토어의 개발자 프로그램 정책을 준수합니다. 본 개인정보처리방침은 확장 프로그램이 사용자의 데이터를 어떻게 취급하는지 설명합니다.

## 1. 수집하는 정보
본 확장 프로그램은 사용자를 식별할 수 있는 어떠한 개인정보(이름, 이메일, IP 주소 등)도 수집, 저장, 또는 외부 서버로 전송하지 않습니다.

확장 프로그램은 다음과 같은 데이터를 **오직 사용자의 브라우저 내부(로컬 환경)에서만** 처리합니다:
- **웹사이트 콘텐츠:** 사용자가 명시적으로 선택한 활성화된 탭의 텍스트, 이미지 주소, 하이퍼링크 등의 DOM 요소.
- **설정 데이터:** 사용자가 옵션 페이지에서 설정한 내용 (예: 번역 언어, Gemini API 키 등). 이 데이터는 브라우저의 로컬 스토리지(`chrome.storage`)에 안전하게 저장되며 외부 서버로 전송되지 않습니다.

## 2. 데이터의 사용 목적
- **마크다운 변환:** 추출된 웹사이트 콘텐츠는 마크다운(.md) 포맷으로 변환되어 사용자 기기에 파일로 다운로드하기 위한 목적으로만 사용됩니다.
- **AI 번역 기능 (선택 사항):** 사용자가 선택적으로 AI 번역 기능을 활성화하고 직접 API 키를 입력한 경우에 한해, 추출된 텍스트 데이터가 Google의 Gemini API 서버로 전송되어 번역 처리에 사용될 수 있습니다. 이 과정은 전적으로 사용자의 동의와 명시적 설정 하에 이루어집니다.

## 3. 제3자 제공 및 공유
본 확장 프로그램은 수집된 사용자 데이터를 제3자에게 판매, 임대, 또는 공유하지 않습니다.

## 4. 데이터 보안
확장 프로그램이 처리하는 모든 웹페이지 데이터는 사용자 기기의 로컬 환경에서만 즉시 처리되며, 별도의 백엔드 데이터베이스에 보관되지 않습니다.

## 5. 개인정보처리방침의 변경
본 개인정보처리방침은 기능 추가나 정책 변경에 따라 업데이트될 수 있으며, 중대한 변경이 있을 경우 스토어 페이지 등을 통해 안내해 드립니다.
